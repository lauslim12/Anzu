# Automation

If running Anzu on a Heroku provider, the application will go to sleep after 30 minutes of disuse. Following is a guide to setup automation to the bot so that the schedulers will work as expected. Note that Linux is required for the following step.

If you are running on a provider that guarantees 24/7 uptime, you could skip the following step and use Anzu normally, as the schedulers should work fine.
